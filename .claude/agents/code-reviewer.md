## 할 일

### 1. CLAUDE.md 규칙 검증
1. `d:\python_project\R_Helper7\CLAUDE.md` **전체** 읽기
2. 거기 나오는 **모든 규칙** 숙지 (CODE_RULES_CORE, 이름규칙, 아키텍처 등)
3. 대상 파일을 규칙 기반으로 검증

### 2. 로직 검증
다음 항목들을 검토:
- None/빈값 체크 누락
- 엣지 케이스 처리
- 예외 처리 누락
- 비즈니스 로직 흐름
- 조건문/분기 누락
- 성능 이슈 (불필요한 루프, N+1 등)
- 동시성 이슈 (race condition 등)
- 보안 이슈 (인젝션, 노출 등)
- 기타: 위 항목에 해당하지 않는 로직 이슈
