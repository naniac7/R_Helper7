# Site Crawler 테스트 체크리스트

## 수동 테스트 시나리오

### 실행
```bash
python -m src.features.site_crawler
```

### 체크리스트

- [ ] **주소 검색**: 주소 입력 → 자동완성 목록 표시 확인
  - 테스트 주소 예시: "서울시 강남구 테헤란로"
  - 자동완성 목록이 콤보박스에 표시되는지 확인

- [ ] **주소 선택**: 주소 선택 → 건물 목록 표시 확인
  - 자동완성 목록에서 주소 하나 선택
  - 건물 목록이 건물 선택 콤보박스에 표시되는지 확인

- [ ] **건물 선택 및 크롤링**: 건물 선택 → 상세 정보 크롤링 → JSON 저장 확인
  - 건물 하나 선택
  - 크롤링 행에 결과가 표시되는지 확인
  - 파일 경로: `data/results/latest_crawl.json` 생성 확인

- [ ] **프리셋 저장**: 크롤링 행 제목 입력 → 프리셋 저장 → `data/presets/crawl_presets.json` 확인
  - 크롤링 행에 제목 입력 (예: "전용면적", "층수", "매매가")
  - 메뉴바 → 설정 → 프리셋 저장하기 클릭
  - 파일 경로: `data/presets/crawl_presets.json` 생성 확인

- [ ] **프리셋 불러오기**: 프리셋 불러오기 → 크롤링 행에 제목 자동 입력 확인
  - 메뉴바 → 설정 → 프리셋 불러오기 클릭
  - 저장된 제목들이 크롤링 행에 자동으로 입력되는지 확인

- [ ] **헤드리스 모드**: 메뉴바 → 헤드리스 모드 토글 → 재실행 시 적용 확인
  - 메뉴바 → 설정 → 헤드리스 모드 체크
  - 프로그램 종료 후 다시 실행
  - 브라우저 창이 표시되지 않는지 확인 (백그라운드 실행)

## 예상 결과

모든 체크리스트 항목이 정상 동작하면 리팩토링 성공입니다.

## 주요 변경사항 확인

### 이벤트 시스템
- 콘솔에 상태 메시지가 정상적으로 표시되는지 확인
- 에러 발생 시 `[에러]` 접두사와 함께 메시지가 표시되는지 확인

### 레이어 구조
- 새로운 폴더 구조가 정상적으로 동작하는지 확인:
  - `domain/`: 비즈니스 엔티티 및 이벤트
  - `app/`: 유즈케이스 및 이벤트 버스
  - `api/gui/`: GUI 컴포넌트
  - `infra/`: Selenium 크롤러 및 파일 저장소

## 문제 해결

### Chrome 드라이버 에러
- ChromeDriver 버전과 Chrome 브라우저 버전이 일치하는지 확인
- `webdriver-manager`가 자동으로 처리하지만, 문제가 있으면 수동 다운로드 필요

### 모듈 import 에러
- Python 경로 설정 확인: `PYTHONPATH`에 프로젝트 루트가 포함되어야 함
- VSCode에서는 `.vscode/settings.json`의 `python.autoComplete.extraPaths` 확인
